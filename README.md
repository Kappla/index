## NVM

* Close terminal
* Reopen it

```bash
# test if NVM is installed correctly
nvm version

# install NodeJS
nvm install 12.16.1

# test NodeJS
node --version
```

## Yarn

```bash
brew install yarn

# test if yarn was installed sucessfully
yarn --version
```
